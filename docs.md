# Документация: Лендинг «Челлендж: 1 навык за 7 дней»

## Состав
- `index.html` — структура страницы, Bootstrap 5, навигация, разметка секций.
- `styles.css` — кастомные стили, мобильная типографика, hover‑правила только для курсора.
- `script.js` — валидация формы в реальном времени и при отправке.
- Скриншоты: сделай 3 скрина после открытия `index.html` локально или на хостинге:
  1) Навбар в мобильном виде (ширина 375px).  
  2) Hero-секция на мобильном.  
  3) Секция преимуществ + форма на десктопе.

## Как запустить
1. Открой `index.html` в браузере. Интернет нужен для CDN Bootstrap и шрифтов.
2. Проверь адаптив: сожми окно до ~375px, затем расширь до планшета и десктопа.

## Этап 1: Улучшения для мобильных
- Типографика: `h1/display-5` уменьшены до ~28px на <576px (`1.75rem`), `lead` = 16px. Читаемо без зума.
- Изображения: `img{max-width:100%;height:auto}` и фоновая картинка через `<picture>` с mobile‑src.
- Навигация: бургер + крупная CTA, `min-height:44px` у интерактивных элементов.
- Отступы: увеличены вертикальные паддинги в hero, крупные поля формы `.form-control-lg`.
- Тач‑цели: переменная `--touch-min:44px` применяется к кнопкам и полям.

## Этап 2: Навбар (Bootstrap 5)
- Логотип слева, ссылка на `#top`.
- Бургер‑меню на мобильных с плавной анимацией по умолчанию от Bootstrap.
- Пункты: Главная/О нас/Услуги/Контакты, плюс CTA справа.
- `sticky-top` + тёмная тема `navbar-dark bg-dark` для контраста.

## Этап 3: Hero для мобильных
- Заголовок краткий, помещается в 1–2 строки. Подзаголовок короче.
- Кнопка `btn-lg` высотой ≥44px. Кнопка на мобильных на всю ширину.
- Фоновое изображение без мелких деталей, кадрирование `object-fit:cover`.
- Паддинги по брейкпоинтам для «воздуха».

## Этап 4: Адаптивная сетка преимуществ
- Bootstrap grid: `col-12` (мобайл 1 колонка) / `col-md-6` (планшет 2) / `col-lg-4` (десктоп 3).
- Карточки с иконкой, заголовком и текстом. Hover‑эффект активен только при `(hover:hover) and (pointer:fine)`.

## Этап 5: Контактная форма
- Поля и кнопка на всю ширину, `.form-control-lg` и `.btn-lg` для тач‑целей ≥44px.
- `inputmode` и `type` подбирают клавиатуру: email, tel, text.
- Валидация в реальном времени: красные подсказки через `.invalid-feedback`.
- Сабмит имитирован `alert`. Для бэкенда заменить на `fetch` POST.

## Доступность (a11y)
- Контрастная тёмная навигация. Чёткие фокусы Bootstrap.
- Лэйблы привязаны к полям. Подсказки об ошибках через `invalid-feedback`.

## Как сделать скриншоты
- Открой DevTools → Toggle device toolbar → выбери iPhone 12 или ширину 375px.
- Сделай 2 скрина мобильных секций и 1 десктопный.
- Сохрани как `screen-1-mobile.png`, `screen-2-mobile.png`, `screen-3-desktop.png`.

## Развёртывание
- Залить папку на любой статический хостинг: GitHub Pages, Netlify, Vercel, Tilda Zero‑блоки.
